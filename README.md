# 📅 Interactive Calendar Component

## 📌 Overview

This project is a responsive and interactive wall calendar component built using React. It is inspired by a physical wall calendar design with a hero image, date grid, and notes section.

## ✨ Features

* 📆 Day range selection (start → end)
* 📝 Notes system (with delete option)
* 💾 Data persistence using localStorage
* 🎨 Wall calendar UI with hero image
* 🎉 Holiday indicators
* 📱 Fully responsive (mobile + desktop)
* 🔴 Today highlight

## 🛠️ Tech Stack

* React (Frontend)
* HTML, CSS, JavaScript

## ⚙️ How to Run Locally

1. Clone the repository:

```
git clone https://github.com/your-username/interactive-calendar.git
```

2. Navigate to the folder:

```
cd interactive-calendar
```

3. Install dependencies:

```
npm install
```

4. Run the project:

```
npm start
```

5. Open in browser:

```
http://localhost:3000
```

## 🎯 Design Decisions

* Used React state for managing date selection and notes
* localStorage used instead of backend (as per requirements)
* Clip-path and layout used to mimic wall calendar design

## 🚀 Future Improvements

* Add animations (flip/transition)
* Add drag selection
* Add theme switching
